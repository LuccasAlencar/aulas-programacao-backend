

# Aula 1 — O que é um projeto?

## 🎯 Objetivos da aula

Ao final desta aula você deverá ser capaz de:

- entender o que é um **projeto**
- diferenciar **projeto** de **atividade rotineira**
- identificar projetos no dia a dia

---

- trabalho em grupo
- construção de algo
- organização de um evento

Anote um exemplo.

---

## 📚 Parte 2 — O que é um projeto? (10 min)

Um **projeto** é um esforço **temporário** realizado para criar **algo único**.

Um projeto possui três características principais:

- tem **início**
- tem **fim**
- tem um **objetivo**

### Exemplos de projetos

| Projeto | Resultado |
|------|------|
Criar um aplicativo | App funcionando |
Desenvolver um site | Site publicado |
Construir uma casa | Casa pronta |
Organizar um evento | Evento realizado |

---

## 🔎 Projeto vs Rotina (10 min)

Nem toda atividade é um projeto.

| Projeto | Rotina |
|------|------|
Tem início e fim | Acontece sempre |
Cria algo novo | Mantém algo existente |
Tem objetivo específico | É repetitiva |

### Exemplo

Projeto  
→ Criar um sistema de cadastro

Rotina  
→ Usar o sistema todos os dias
