# Aula 6 — Definindo o problema do projeto

## 🎯 Objetivos

Você aprenderá a:

- definir um problema corretamente
- estruturar desafios de projeto
- preparar o desenvolvimento de soluções

---

## ❗ Por que definir bem o problema? (10 min)

Se o problema estiver errado:

- a solução será ruim
- o projeto falhará

Por isso precisamos **definir claramente o problema**.

---

## 🧩 Estrutura de um bom problema (10 min)

Um problema bem definido responde:

- **Quem** tem o problema
- **Qual** é o problema
- **Quando** acontece
- **Impacto**

---

### Exemplo

Problema ruim:

> alunos esquecem coisas.

Problema bem definido:

> alunos do ensino médio esquecem prazos de atividades porque não possuem um sistema organizado de lembretes.

## 🚀 Desafio final

Transforme o problema em uma pergunta:

Exemplo:

> Como podemos ajudar alunos a lembrar suas tarefas escolares?

Agora escreva a sua.
