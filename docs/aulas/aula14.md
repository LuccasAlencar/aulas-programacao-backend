# Aula 5 — Entendendo o usuário (Empatia)

## 🎯 Objetivos

Nesta aula você irá:

- entender o conceito de **empatia**
- identificar necessidades dos usuários
- analisar problemas reais

---

## 👥 O que é empatia? (10 min)

Empatia significa:

> entender o problema do ponto de vista de outra pessoa.

Perguntas importantes:

- quem tem o problema?
- quando acontece?
- como a pessoa se sente?

---

## 🔎 Exemplo (10 min)

Problema:

> alunos esquecem prazos de atividades.

Perguntas de empatia:

- quando isso acontece?
- por que acontece?
- como os alunos se sentem?
